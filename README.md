## Login Page
This React-based web app allows users to login and logout of the application with different validations.

## Technologies Used
- React 17.0
- Nodejs
- CSS
- HTML
- Visual Studio Code

## Concepts Used
- useEffect()
- useReducer()
- useContext()
- useRef()
